# SmartStart
This folder regroups the specifications and application notes needed for developing SmartStart nodes.

For Z-Wave Plus nodes, requirements are specified in the following specifications:
* SDS11847 Z-Wave Plus Device Type Specification: section 3.10 SmartStart requirements
* SDS11846 Z-Wave Plus Role Type Specification:   section 3.12 SmartStart requirements

For Z-Wave Plus v2 nodes, requirements are specified in the following specifications:
* SDS14224 Z-Wave Plus v2 Device Type Specification: section 3.8 SmartStart requirements
* SDS11846 Z-Wave Plus Role Type Specification:   section 3.12 SmartStart requirements
